# 📈 Market Glitch - Pro Trading Game

Ein minimalistisches High-Speed Trading Spiel für zwischendurch. 
Meistere den Markt, gehe Long oder Short und schalte neue Level frei!

## 🎮 Steuerung
- **Linke Bildschirmhälfte:** LONG halten (Gewinn bei steigendem Kurs)
- **Rechte Bildschirmhälfte:** SHORT halten (Gewinn bei fallendem Kurs)
- **Ziel:** Erreiche $5000 für das nächste Level!

## 📱 Installation
Dies ist eine PWA (Progressive Web App). Öffne den Link am Handy und wähle "Zum Home-Bildschirm hinzufügen", um es als App zu speichern.
